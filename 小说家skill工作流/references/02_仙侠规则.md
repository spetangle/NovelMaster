# 仙侠题材创作规则

> 版本: 1.0 | 题材: 仙侠 | 适用: 修真、飞升、道法

---

## 核心特征

- **力量体系**: 金丹、元婴、化神、飞升
- **主角定位**: 求道者，逆天改命
- **核心爽点**: 顿悟突破、炼丹炼器、宗门崛起
- **固定套路**: 坠崖得宝、仙人遗府、丹药灵根

---

## 章节结构模板

### 标准修仙章节（6000字）

```
【章节节奏】
第1段 (0-800字):   日常/修炼 → 铺垫
第2段 (800-2400字): 奇遇/顿悟 → 核心
第3段 (2400-4000字): 突破/战斗 → 高潮
第4段 (4000-5200字): 收获/宗门 → 格局
第5段 (5200-6000字): 伏笔/劫难 → 钩子
```

---

## 力量等级体系

### 标准仙侠等级（20级）

```
炼气 → 筑基 → 金丹 → 元婴 → 化神 → 合道 → 大乘 → 渡劫 → 地仙 → 天仙 → 金仙 → 太乙 → 大罗 → 道祖 → 天道
```

### 境界差距战力标准

| 境界差 | 战斗结果 |
|--------|----------|
| 小境界（3级内） | 可越级挑战 |
| 大境界（5级+） | 几乎无法战胜 |
| 越大境界 | 需特殊手段/宝物 |

---

## 核心资源体系

| 资源 | 用途 | 获取难度 |
|------|------|----------|
| 灵气/灵力 | 修炼根基 | 中 |
| 灵根 | 资质决定上限 | 难（天生） |
| 丹药 | 辅助突破 | 中 |
| 法宝 | 战力倍增 | 难 |
| 功法 | 决定成长速度 | 难 |
| 灵宠/坐骑 | 辅助战力 | 中 |

---

## 仙侠爽点场景

```
1. 顿悟突破 → 三年一小悟，十年一大悟
2. 炼丹成功 → 九转金丹，天降异象
3. 遗迹探险 → 上古仙人洞府
4. 宗门大比 → 天骄云集，一战成名
5. 灵宠认主 → 神兽幼崽，主动追随
```

---

## 宗门体系模板

```
门派层级: 外门弟子 → 内门弟子 → 真传弟子 → 长老 → 掌门
资源等级: 凡级 → 灵级 → 宝级 → 仙级 → 神级
势力分布: 世俗皇朝 + 修仙宗门 + 隐秘势力 + 远古遗迹
```

---

## 开篇黄金三章

```
第一章: 主角现状（废灵根/被欺/家破）
     + 金手指（老爷爷/系统/重活一世）
     + 第一个机遇（坠崖/捡宝/偶得功法）

第二章: 初次修炼 + 小试身手
     + 势力格局展示

第三章: 宗门/学院入门
     + 埋下主线伏笔（劫难/阴谋）
```

---

## 禁用俗套

```
✗ 无脑升级（需瓶颈+顿悟机制）
✗ 红雾灭世（增加阴谋/气运争夺）
✗ 单刷副本（增加团队协作元素）
✗ 女主花瓶（增加女主独立成长线）
```

• 灵宠/法宝需有来历说明
