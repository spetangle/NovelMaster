# 玄幻题材创作规则

> 版本: 1.0 | 题材: 玄幻 | 适用: 异世穿越、魔法斗气、升级流

---

## 核心特征

- **力量体系**: 等级森严（青铜/白银/黄金... 或 炼体/筑基/金丹...）
- **主角定位**: 弱变强，逆袭打脸
- **核心爽点**: 越级挑战、以弱胜强、装逼打脸
- **固定套路**: 退婚、废柴觉醒、秘境探险、宗门崛起

---

## 章节结构模板

### 标准爽点章节（6000字）

```
【章节节奏】
第1段 (0-800字):   困境/挑衅 → 积蓄情绪
第2段 (800-2400字): 压抑/屈辱 → 等待爆发
第3段 (2400-4000字): 逆转/打脸 → 高潮释放
第4段 (4000-5200字): 收获/装逼 → 巩固胜利
第5段 (5200-6000字): 伏笔/悬念 → 钩子收尾
```

### 关键场景公式

| 场景类型 | 公式 | 目的 |
|----------|------|------|
| 越级战斗 | 轻视 → 被打脸 → 震惊 → 跪服 | 建立主角威信 |
| 宝物获取 | 危机 → 突破 → 收获 | 推进剧情+展示奇遇 |
| 势力展示 | 嘲讽 → 底蕴曝光 → 敌方恐惧 | 拉高格局 |
| 境界突破 | 积累 → 瓶颈 → 顿悟 → 爆发 | 成长线节奏 |

## 开篇黄金三章规范

### 第一章必含要素

```
1. 世界观速览（300字内）
2. 主角现状+困境（弱/被欺/资源匮乏）
3. 金手指曝光（系统/老爷爷/前世记忆）
4. 第一个冲突（挑衅/阴谋/危机）
5. 钩子（留悬念，引导继续阅读）
```

### 第二章爽点节奏

```
第1个爽点: 小型逆转（打脸配角）
第2个爽点: 金手指效果展示
伏笔: 埋下更大的局
```

### 第三章格局拉升

```
扩大冲突范围（家族→宗门/学院）
引入更强的对手
暗示主线危机
```

## 力量等级体系规范

### 等级命名模板（任选其一）

**模板A - 简化版（15级）**
```
凡人 → 炼体境（1-3）→ 炼气境（4-6）→ 筑基境（7-9）→ 金丹境（10-12）→ 元婴境（13-15）→ 大乘 → 渡劫 → 仙人
```

**模板B - 西方奇幻版**
```
平民 → 战士/学徒 → 青铜 → 白银 → 黄金 → 铂金 → 钻石 → 史诗 → 传说 → 神级
```

### 战斗力差距标准

| 等级差 | 战斗结果 | 描述 |
|--------|----------|------|
| 1级 | 艰难取胜 | 险胜 |
| 2级 | 可战胜 | 略占上风 |
| 3级 | 碾压 | 轻松取胜 |
| 5级+ | 无法战胜 | 天堑 |

## 打脸场景标准流程

```
1. 铺垫（让读者积累愤怒）
   - 反派嚣张言行
   - 旁观者嘲讽
   - 主角"无能"证据

2. 转折（制造信息差）
   - 反派/旁观者不知道的底牌曝光
   - 主角真实身份/实力展示

3. 高潮（情绪爆发）
   - 反派震惊/恐惧
   - 围观者惊呼
   - 主角云淡风轻

4. 收尾（巩固爽感）
   - 反派下场
   - 奖励获取
   - 伏笔铺垫
```

## 升级流核心节点

| 阶段 | 字数节点 | 内容要求 |
|------|----------|----------|
| 入门 | 10万字内 | 确立金手指，快速升级 |
| 小成 | 30万字内 | 走出新手村，进入大势力 |
| 中成 | 50万字内 | 接触核心主线 |
| 大成 | 80万字内 | 决战准备 |
| 结局 | 100万字+ | 最终BOSS战 |

## 禁用俗套（高级写法要求）

```
✗ 退婚必须跪舔（改为平等契约或女主强势）
✗ 反派降智（给反派合理动机）
✗ 升级全靠嗑药（增加顿悟/战斗领悟）
✗ 无脑爽（增加角色内心成长线）
```

## 章节标题规范

```
格式: [爽点类型]+[核心事件]

示例:
- 「第三章 废柴觉醒，一拳碾压」
- 「第七章 秘境奇遇，逆天改命」
- 「第十二章 宗门大比，横扫天才」
```
