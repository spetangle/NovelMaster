# 题材规则集总索引

> 版本: 1.1 | 整理日期: 2026-05-12

---

## 用途

本目录收录所有题材类型的独立规则文档。Agent 在书籍初始化时**按需读取**，而非一次性加载全部。

## 调用规范

### 规则 A：已知题材初始化

```
1. 用户提交创作简报，注明题材类型
2. Agent 读取对应题材文档（如：02_仙侠规则.md）
3. 将题材规则注入 story_bible.md 与 book_rules.md
4. 禁止加载非指定题材的规则文档
```

### 规则 B：已知题材检索

> 当用户提交创作简报时，先检索题材类型参考文件夹。若题材与现有文档完全一致，直接使用；若存在近似但不完全匹配，提示用户确认。

| 关键词 | 题材类型 | 对应文档 |
|--------|----------|----------|
| 修炼、飞升、金丹、境界、渡劫 | 仙侠 | 02_仙侠规则.md |
| 魔法、斗气、升级、异世、穿越 | 玄幻 | 01_玄幻规则.md |
| 现代、都市、商战、职场、神医、兵王 | 都市 | 03_都市规则.md |
| 星际、科技、异能、赛博机甲 | 科幻 | 04_科幻规则.md |
| 青春、校园、高中、大学、学业 | 青春校园 | 06_青春校园规则.md |
| 其他 | 自定义 | → 触发规则 C |

### 规则 C：自定义题材创建

> 当用户填写的题材不在四大主流类型内时，触发 `genre create` 工作流引导用户创建新题材。

```
1. 用户填写「题材: [非标准类型]」
2. Agent 读取 05_自定义规则.md
3. 引导用户补充核心设定、规则体系
4. 生成该书专属的 [编号]_[题材名]规则.md
5. 更新本索引文件
6. 衔接进入 book create 流程
```

**详细工作流**: 参见 `workflow-genre-create.md`
