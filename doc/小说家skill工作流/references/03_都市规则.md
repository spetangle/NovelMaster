# 都市题材创作规则

> 版本: 1.0 | 题材: 都市 | 适用: 现代背景、商战、异能、职场

---

## 核心特征

- **力量体系**: 金钱、权势、人脉、异能
- **主角定位**: 逆袭者/王者归来/普通人崛起
- **核心爽点**: 打脸、赚钱、权力、爱情
- **固定套路**: 退婚流、兵王回归、继承遗产、系统流

---

## 章节结构模板

### 标准都市爽文（6000字）

```
【章节节奏】
第1段 (0-800字):   日常/冲突 → 铺垫
第2段 (800-2400字): 打脸准备 → 积蓄
第3段 (2400-4000字): 真相曝光 → 高潮
第4段 (4000-5200字): 收获/装逼 → 巩固
第5段 (5200-6000字): 伏笔/新局 → 钩子
```

## 都市力量体系

### 社会资源层级

```
底层 → 中产 → 上流 → 顶级权贵 → 隐藏世家/古武/异能势力
```

### 主角金手指类型

| 类型 | 表现 | 爽点 |
|------|------|------|
| 财富系统 | 资产增值/首富系统 | 赚钱爽 |
| 医术/技能 | 神医/黑客/全能 | 能力爽 |
| 身份隐藏 | 兵王/少主/继承人 | 身份爽 |
| 人脉系统 | 大佬追随/兄弟义气 | 势力爽 |

## 核心爽点场景

```
1. 身份打脸 → "你是谁？""我叫..."
2. 财富展示 → 随手消费/收购公司
3. 女人嫉妒 → 前女友/绿茶后悔
4. 权力碾压 → "叫你们老板来"
5. 能力震惊 → 解决别人解决不了的问题
```

## 都市文禁忌（高级写法）

```
✗ 无脑打脸（给反派合理动机）
✗ 女主花瓶（增加女主独立事业线）
✗ 仇富心态（主角消费观正常）
✗ 脱离现实（地名/职业描写需考据）
```
