# InkOS 题材规则集

## 一、玄幻题材规则

### 1.1 数值系统

| 等级 | 名称 | 战力区间 | 说明 |
|------|------|----------|------|
| Lv1-Lv9 | 凡人境 | 1-99 | 普通人类极限 |
| Lv10-Lv19 | 灵徒境 | 100-999 | 踏入修炼门槛 |
| Lv20-Lv29 | 灵师境 | 1000-9999 | 小有所成 |
| Lv30-Lv39 | 灵宗境 | 10000-99999 | 一方强者 |
| Lv40-Lv49 | 灵王境 | 10万-99万 | 区域霸主 |
| Lv50-Lv59 | 灵皇境 | 100万-999万 | 顶尖战力 |
| Lv60-Lv69 | 灵帝境 | 1000万-9999万 | 传说级 |
| Lv70+ | 准神/神境 | 1亿+ | 作品上限 |

### 1.2 同质吞噬衰减公式

> 同类能力效果随使用次数递减，防止无限叠加

```
实际效果 = 理论效果 × (0.7 ^ 已使用次数)
```

例：第3次使用效果 = 理论效果 × 0.49（衰减51%）

### 1.3 爽点兑现节奏

| 阶段 | 节奏 | 预期爽点 |
|------|------|----------|
| 前期（1-30章） | 打脸+升级并行 | 每3-5章一次小打脸，每10章一次大升级 |
| 中期（31-60章） | 势力博弈+资源争夺 | 打脸升级频率降低，转向团战/商战 |
| 后期（61-80章） | 终极对决+终极伏笔回收 | 最终boss战，一句话秒杀前中期boss |

### 1.4 禁止规则

- 主角战力提升不得超过相邻大境界的50%
- 同一天内最多触发2次打脸事件
- 每次大升级后必须有3章缓冲期

## 二、仙侠题材规则

### 2.1 修炼/悟道节奏

| 境界 | 悟道要求 | 瓶颈类型 |
|------|----------|----------|
| 炼气期 | 吸纳灵气，打通经脉 | 心魔初现 |
| 筑基期 | 筑固根基，丹田中正 | 资质门槛 |
| 金丹期 | 结丹凝婴，元神初成 | 天劫降临 |
| 元婴期 | 元神出窍，神游太虚 | 天道压制 |
| 化神期 | 感悟天道，神人合一 | 情劫缠身 |
| 大乘期 | 渡劫飞升，临门一脚 | 心魔大劫 |
| 渡劫期 | 飞升仙界 | 飞升雷劫 |

### 2.2 法宝体系

| 品阶 | 威力 | 稀有度 | 示例 |
|------|------|--------|------|
| 法器 | 低 | 常见 | 飞剑、符箓 |
| 灵器 | 中 | 稀有 | 玉佩、葫芦 |
| 宝器 | 高 | 罕见 | 青铜古镜、九龙鼎 |
| 道器 | 极高 | 传说 | 天道碎片、命运罗盘 |
| 仙器 | 神级 | 唯一 | 主角专属或最终boss持有 |

### 2.3 天道规则

- 天道压制：修为越高，天道反噬越强
- 因果法则：杀人夺宝必沾因果，需偿还
- 气运法则：主角气运护体，机缘更多

### 2.4 禁止规则

- 禁止越境击杀（除非主角有特殊金手指）
- 法宝认主后不可强行剥夺
- 飞升前必须解决所有尘世因果

## 三、都市题材规则

### 3.1 年代考据要求

| 时代背景 | 考据要点 |
|----------|----------|
| 90年代 | 物价水平、通讯工具（BB机）、流行音乐 |
| 2000年代 | 手机普及、网络文学兴起、汽车进入家庭 |
| 2010年代 | 智能手机、社交媒体、移动支付 |
| 2020年代 | AI元年、直播经济、元宇宙概念 |

### 3.2 驱动模式

| 类型 | 核心冲突 | 爽点来源 |
|------|----------|----------|
| 商战 | 商业竞争、企业收购 | 商业头脑碾压对手 |
| 社交 | 人际关系、家族纷争 | 人脉拓展、打脸势利眼 |
| 都市异能 | 超凡能力、势力洗牌 | 能力碾压、社会地位跃升 |
| 神医/兵王 | 专业领域、扮猪吃虎 | 专业能力打脸 |

### 3.3 法律术语匹配

- 合同法：违约金条款、竞业禁止
- 婚姻法：彩礼、房产归属
- 刑法：正当防卫、故意伤害认定

### 3.4 禁止规则

- 禁止违背现实经济规律（如主角随手中彩票500万）
- 禁止现代人使用超时代词汇（如"内卷"、"躺平"用于90年代背景）
- 禁止忽视社会阶层固化的存在

## 四、科幻题材规则

### 4.1 科技树设定

| 层级 | 技术水平 | 社会形态 |
|------|----------|----------|
| T1 | 核能时代 | 信息社会 |
| T2 | 量子时代 | 星际殖民萌芽 |
| T3 | 暗能量时代 | 银河联邦 |
| T4 | 维度时代 | 多维度文明 |
| T5 | 规则改变者 | 超越物理法则 |

### 4.2 高能状态规则

| 状态 | 触发条件 | 持续时间 | 副作用 |
|------|----------|----------|--------|
| 超频 | 情绪剧烈波动 | 30秒-5分钟 | 体力透支 |
| 觉醒 | 生死危机/情感冲击 | 永久 | 记忆碎片化 |
| 暴走 | 失控/被动触发 | 不定 | 敌我不分 |

### 4.3 异能觉醒机制

```
觉醒概率 = 基础概率 × (精神状态系数) × (血缘纯度系数)
```

- 觉醒后需3-5章适应期
- 每次突破需对应"觉醒契机"

### 4.4 禁止规则

- 禁止科技与修仙混搭（除非作品明确设定）
- 禁止出现超越作品科技树层级的道具
- 禁止异能无代价无限使用

## 五、其他/自定义题材规则

### 5.1 异能觉醒类

| 觉醒类型 | 特征 | 成长曲线 |
|----------|------|----------|
| 血统觉醒 | 家族遗传 | 阶梯式爆发 |
| 契约觉醒 | 外部力量 | 被动触发为主 |
| 自我觉醒 | 内心突破 | 线性成长 |

### 5.2 宇宙高能狩猎族群

| 种族 | 核心能力 | 社会结构 |
|------|----------|----------|
| 狩猎族 | 追踪、围猎 | 部落制 |
| 星舰族 | 星际航行、舰载武器 | 帝国制 |
| 融合族 | 吸收、同化 | 蜂巢制 |
| 机械族 | 改造、进化 | 军阶制 |

### 5.3 自定义规则模板

> 当题材不属于以上四类时，使用此模板

```markdown
## [自定义题材] 特殊规则

### 核心设定
[描述该题材的核心世界观]

### 能力体系
[描述主角及配角的特殊能力]

### 成长节奏
[描述主角成长的时间线]

### 禁止事项
- 禁止[规则1]
- 禁止[规则2]
```

## 六、题材识别流程（Agent 01/03 共用）

```
输入: 用户创作简报
    │
    ▼
[题材识别]
    │
    ├─ 含"修炼""飞升""金丹" → 仙侠
    ├─ 含"都市""现代""职场" → 都市
    ├─ 含"星际""科技""异能" → 科幻
    ├─ 含"魔法""斗气""升级" → 玄幻
    └─ 其他 → 自定义（需补充规则）
    │
    ▼
输出: 题材类型 + 对应规则集
```
