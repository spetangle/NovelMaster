# 科幻题材创作规则

> 版本: 1.0 | 题材: 科幻 | 适用: 星际、机甲、赛博、异能、末日

---

## 核心特征

- **力量体系**: 科技、异能、基因进化
- **主角定位**: 星际开拓者/异能觉醒者/文明守护者
- **核心爽点**: 科技碾压、星际探险、异能觉醒
- **固定套路**: 陨石灾变、星际战争、机甲对决、AI觉醒

---

## 章节结构模板

### 标准科幻爽文（6000字）

```
【章节节奏】
第1段 (0-800字):   世界观展示 → 铺垫
第2段 (800-2400字): 危机/机遇 → 核心
第3段 (2400-4000字): 科技/异能展示 → 高潮
第4段 (4000-5200字): 收获/格局 → 拉升
第5段 (5200-6000字): 伏笔/更大危机 → 钩子
```

## 科幻力量体系

### 科技等级（卡尔达肖夫指数）

```
0型 → 行星文明 → 恒星文明 → 星系文明 → 宇宙文明
```

### 主角能力类型

| 类型 | 代表 | 爽点 |
|------|------|------|
| 科技流 | 机甲/战舰/AI | 装备碾压 |
| 异能流 | 精神/肉体/元素 | 觉醒爆发 |
| 进化流 | 基因变异/融合 | 生命跃迁 |
| 混合流 | 科技+异能 | 综合优势 |


## 科幻文禁忌

```
✗ 科技BUG（设定需自洽）
✗ 无脑战斗（策略与配合）
✗ 忽视代价（高科技需高消耗）
✗ 人类中心主义（宇宙观需包容）
```
