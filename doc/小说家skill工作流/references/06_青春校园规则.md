> 题材: 自定义
> 版本: 1.0
> 参考类型: 现代都市校园（无异能/无超自然）
> 整理日期: 2026-05-12

---

## 一、世界观设定

### 1.1 时代背景

**时间线**：现代（中国/东亚教育体系语境）
**空间范围**：重点高中/重点大学为主场景

- 核心场景：教室、校园、社团、家庭
- 扩展场景：补习班、竞赛场地、社团活动、社会实践
- 时代特征：信息化时代、手机普及、考试升学压力

### 1.2 核心规则

**校园法则**
- 学业成绩作为核心衡量标准
- 升学考试（中考/高考/考研）作为阶段性目标节点
- 社团活动提供能力成长与社交突破的副线

**社会法则**
- 家庭背景影响初始资源与社交起点
- 人际关系影响信息获取与机会分配
- 阶层差异形成潜在冲突与逆袭动机

**禁止元素**：异能/超自然/魔法/修仙体系全部禁止

### 1.3 校园体系

```
学业等级（参照现实，但不写死）：
- 学渣（排名后30%）
- 普通学生（前50%）
- 优等生（前20%）
- 学霸/学神（前5%）
- 竞赛选手/保送生（特殊通道）

社交等级：
- 透明人（小透明）
- 普通同学
- 班级干部/课代表
- 风云人物
- 社团核心/学生干部
```

## 二、能力体系

### 2.1 能力来源

**学习能力**（核心）
- 记忆力、理解力、分析力、总结能力
- 学习方法论：笔记、错题本、时间管理
- 刻意练习：专项突破、题海战术

**天赋加成**（辅助）
- 智商天赋（过目不忘、逻辑强、数学直觉）
- 情商天赋（社交直觉、情绪控制）
- 体能天赋（精力充沛、睡眠需求少）

**资源加持**（辅助）
- 家庭背景：有钱/有权/有教育背景
- 外部支持：名师辅导、优质教材、信息渠道
- 人脉网络：学长学姐、同学关系、师生关系

### 2.2 能力成长路径

```
主线：学业逆袭路径
学渣觉醒 → 找到方法 → 刻意练习 → 成绩突破 → 打脸碾压 → 稳定高位 → 下一阶段

辅线：社交逆袭路径
透明人 → 展示价值 → 吸引关注 → 建立人设 → 成为风云人物

辅线：竞赛/特长路径
发现特长 → 专项训练 → 参赛获奖 → 获得保送/加分
```

### 2.3 学业能力量化（隐性）

> 注：青春校园题材不写明确数值，但作者需清楚主角的「相对位置」变化

```
起点：班级排名后30%
中期：班级前50%
后期：年级前20%
终期：竞赛级别/保送资格
```

## 三、章节节奏

| 阶段 | 章节范围 | 节奏特征 | 核心爽点 |
|------|----------|----------|----------|
| 觉醒期 | 1-10章 | 缓慢铺垫，情绪压抑 | 顿悟时刻、被轻视后暗下决心 |
| 突破期 | 11-25章 | 逐步升温，积累期 | 方法验证、排名提升、小试牛刀 |
| 高光期 | 26-40章 | 集中爆发，打脸密集 | 大考逆袭、碾压嘲讽者、获得认可 |
| 稳定期 | 41-60章 | 新挑战新目标 | 建立人设、社交突破、特长发展 |

### 3.1 单章节奏模板

```
起点：困境/冲突（500-800字）
积累：学习方法/努力过程（1500-2000字）
转折：契机/顿悟（500-800字）
结果：成果展示/打脸铺垫（1000-1500字）
收尾：爽感余韵/伏笔留钩（500字）

总字数：约4000-5000字/章（都市校园规格）
```

## 四、爽点设计

### 4.1 核心爽点类型

**学业逆袭爽点**
- 「学渣→学霸」的蜕变叙事
- 考试进步：班级排名提升、年级排名提升
- 竞赛获奖：学科竞赛、作文大赛、创新大赛
- 升学成功：中考/高考/保送/留学

**打脸碾压爽点**
- 当众打脸：成绩公布时碾压嘲讽者
- 场景打脸：家长会/班会/竞赛现场逆转
- 持续打脸：多次考试连续碾压，形成威慑

**社交逆袭爽点**
- 从透明人到班级风云人物
- 获得异性关注/好感
- 成为社团核心/学生干部
- 建立自己的小团队/圈子

**情感爽点**
- 青涩暗恋的酸甜
- 双向奔赴的甜蜜
- 误会导致的虐心
- 和解后的升温

### 4.2 爽点兑现模板

**模板A：考试逆袭打脸**
```
场景：月考/期中考/期末考成绩公布
标准结构：
1. 铺垫轻视（老师念成绩/排名下滑者被嘲讽）
2. 主角被点名（众人期待主角出丑）
3. 成绩揭晓（从低到高报，主角名字最后出现）
4. 真相揭露（主角实际是班级/年级前XX名）
5. 打脸反应（嘲讽者震惊/老师刮目相看/同学议论）
6. 余韵钩子（暗示下次考试还有更大目标）
```

**模板B：社团/特长逆袭**
```
场景：运动会/文艺汇演/学科竞赛
标准结构：
1. 主角被轻视（被认为只是成绩好/其他方面不行）
2. 意外机会（替补/临时报名/被激将）
3. 全力准备（方法论展示/努力过程）
4. 高光时刻（比赛/表演中的精彩表现）
5. 结果确认（获奖/获得认可）
6. 社交影响（人设建立/关系变化）
```

**模板C：情感升温**
```
场景：日常相处/特定事件/情感危机
标准结构：
1. 日常互动（借笔记/一起值日/讨论问题）
2. 特殊事件（对方遇到困难/情绪波动）
3. 主角介入（提供帮助/陪伴/安慰）
4. 关系变化（从普通同学到特殊关系）
5. 甜蜜时刻（对方的小动作/暗示好感）
6. 钩子留白（下一个接触机会/潜在冲突）
```

### 4.3 爽点禁忌

- 禁止「系统」/「异能」等超自然元素
- 禁止「一朝顿悟后天下无敌」的无脑爽
- 禁止「一次性打脸后永远碾压」的机械重复
- 打脸后必须有**代价**或**新的挑战**，避免爽感断层
- 情感线必须与**主线目标**绑定，禁止纯恋爱脑

## 五、人物设定规范

### 5.1 主角定位

**标准模板**：
- 初始状态：学渣/被轻视/有隐藏天赋
- 核心动机：证明自己/守护重要的人/改变命运
- 成长路径：方法论+努力+机遇

**外貌描写频率**：
> 青春校园题材以「代入感」为核心，外貌描写需克制
- 开篇：1次完整外貌描写
- 中期：每5-8章1次精简外貌（聚焦某个特征）
- 禁止：每章必写外貌、重复描写相同特征

**外貌描写模板**：
```
精简版：「他穿着洗得发白的校服，眼神专注地盯着题目」
聚焦版：「她扎着马尾，额角沁出细密的汗珠，手指因为紧张而微微发白」
```

### 5.2 配角功能

| 角色类型 | 功能 | 常见定位 |
|----------|------|----------|
| 嘲讽者 | 提供打脸对象 | 绿茶同学/富二代/前学霸 |
| 伙伴 | 陪伴成长 | 死党/闺蜜/志同道合者 |
| 导师 | 提供方法/机遇 | 老师/学长/家长 |
| 目标 | 情感线/竞争线 | 喜欢的人/学习目标 |
| 背景板 | 丰富世界观 | 同学群体/家长群像 |

### 5.3 角色命名规范

- 主角：普通姓名（代入感），避免太中二的名字
- 配角：特征+名字，如「绿茶女配」「富二代同学」
- 禁止：过度中二的名字、奇怪姓氏的组合

## 六、禁忌清单

### 6.1 内容禁区

- **禁止异能/系统/超自然**：青春校园核心是「现实努力」
- **禁止血腥暴力**：校园场景禁止过度暴力描写
- **禁止色情擦边**：恋爱描写止于牵手/暗恋/青涩互动
- **禁止过度拜金**：可以写家庭差距，但不能宣扬炫富

### 6.2 叙事禁区

- **禁止无脑爽**：打脸必须有逻辑支撑，不能凭空碾压
- **禁止学渣人设崩塌**：逆袭必须努力，不能躺赢
- **禁止情感线喧宾夺主**：恋爱是调味剂，主线是学业/成长
- **禁止「一次性永久逆袭」**：每个目标达成后必须有新的挑战

### 6.3 节奏禁区

- **禁止前5章无冲突**：开篇必须有「钩子」
- **禁止连续3章无进展**：每章必须有明确的情节推进
- **禁止爽点断层**：高潮后必须在3章内制造新爽点
