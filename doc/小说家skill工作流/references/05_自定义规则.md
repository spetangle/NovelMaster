# 自定义题材规则模板

> 版本: 1.0 | 题材: 自定义 | 适用: 未知题材/混合题材/创新题材

---

## 用途

当用户填写的题材不在四大主流类型（玄幻/仙侠/都市/科幻）内时，使用本模板引导完善。

---

## 触发条件

```
用户输入: 题材: [非标准类型]
示例: 题材: 游戏同人/军事/历史/悬疑/轻小说/混合
```

---

## 自定义题材补充流程

### 步骤一：核心设定确认

```
请补充以下信息：

1. 世界观基底
   - 时代背景: [古代/现代/未来/架空/异世界]
   - 核心规则: [物理法则/魔法体系/游戏规则]

2. 力量/能力体系
   - 等级形式: [有等级/无等级/隐藏等级]
   - 能力来源: [修炼/科技/天赋/学习/混合]

3. 主角定位
   - 初始状态: [普通人/特殊身份/强者转世]
   - 目标动机: [复仇/探索/守护/征服/其他]
```

### 步骤二：爽点类型确认

```
勾选本书核心爽点（可多选）：
[ ] 升级变强
[ ] 打脸碾压
[ ] 探索揭秘
[ ] 势力扩张
[ ] 情感纠葛
[ ] 技术创新
[ ] 策略博弈
[ ] 其他: _______
```

### 步骤三：生成自定义规则

```
根据以上信息，生成该书专属的 rules.md，内容包含：

1. 世界观设定摘要
2. 力量/能力体系定义
3. 章节节奏建议
4. 爽点场景模板
5. 特有禁忌清单
```

---

## 模板字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| 世界观基底 | 是 | 定义故事发生的基础环境 |
| 能力体系 | 是 | 定义"强"的标准和路径 |
| 爽点类型 | 是 | 明确读者获得爽感的方式 |
| 禁忌清单 | 是 | 防止写歪 |
| 参考题材 | 否 | 如有参考，可填"类似XX+YY" |

---

## 输出格式

```
生成文件: [书名]_rules.md

内容结构:
# [书名] 创作规则
> 题材: 自定义
> 参考类型: [如有]

## 世界观设定
...

## 能力体系
...

## 章节节奏
...

## 爽点设计
...

## 禁忌清单
...
```

---

## 示例

**用户输入**: 题材: 游戏同人-王者荣耀

**生成补充问题**:
```
1. 世界观: 王者荣耀英雄宇宙（已存在）
2. 主角身份: 召唤师/英雄本身/普通人穿越
3. 核心冲突: 守护长安/对抗魔道/英雄内战
4. 爽点: 英雄战斗、技能展示、战队崛起
```

**输出**: `references/自定义_王者荣耀规则.md`
